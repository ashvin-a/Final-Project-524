# CS 524: Final Project
This is the final project that is to be submitted as a standalone application for CS 524 - Introduction to Optimization. I've decided to optimise a logistics problem of a shipping system.

# Setup
To clone this repository, run this command:\
```
git clone https://github.com/ashvin-a/Final-Project-524.git
```

After cloning the repository, create a virtual environment to install the dependencies.\
```
python3 -m venv venv
```

Now activate the environment. If you are using a Linux or Mac system, run\
```
source venv/bin/activate
```

If you are using Windows, run\
```
.\venv\Scripts\activate
```

Now, install the dependencies\
```
pip install -r requirements.txt
```
